## AIによる要約

### 概要と事実
Diditというスタートアップは、双子のエンジニア兄弟が2023年に創業した会社です。AIを活用した本人確認（KYC）と不正検出を、開発者が簡単に組み込める一つのAPIとして提供しています。2026年冬のY Combinatorバッチに参加し、先月27日にシードラウンドで合計750万ドル（今回の追加分は600万ドル）の資金調達を発表しました。資金はグローバル市場の拡大、プログラマブルな本人確認インフラの強化、および営業・開発・カスタマーサクセスの人材採用に使われる予定です。現在、アクティブなB2B顧客は1,500社を超え、提供地域は220カ国・地域に及び、売上は前月比で30％以上成長し、すでに黒字化を達成しています。技術的には、書類の真正性や生体活性、ディープフェイク解析など200以上のシグナルをAIモデルで分析し、人的レビューは全体の約5％にとどまっています。また、スペイン政府から「対面より安全」と認定されるなど、政府データベースとの直接照合も可能です。

### ネットの反応と意見の変遷
記事が公開されると、テクノロジー系のニュースサイトやスタートアップ関連のフォーラムで注目が集まりました。多くのコメントでは、AIを使った迅速な本人確認の仕組みが評価され、「開発者にとって導入が非常に簡単」という点が称賛されました。一方で、プライバシーやデータ保護への懸念も一部で見られ、特に政府データベースとの連携について「個人情報の取り扱いが透明であるか」という議論が行われました。時間が経つにつれ、実際に導入した企業からの成功事例が共有され、特に新規顧客層への浸透が強調されるようになりました。全体としては、導入の手軽さと高いセキュリティへの期待が高まり、批判的な声は次第に少なくなっていった印象です。

### 背景と結論
近年、生成AIの進歩によりディープフェイクや合成IDの作成が容易になり、従来の本人確認手法では対応しきれなくなっています。同時に、EUのeIDAS 2.0など規制が強化され、デジタル身分証の普及が求められています。Diditはこうした環境変化に対応し、多層的な検証アプローチとセルフサーブ型APIを組み合わせることで、開発者やAIエージェントでも簡単に導入できる信頼性の高い基盤を提供しています。スペイン政府の認定や日本市場での早期導入事例は、技術の信頼性と実用性を裏付けるものです。今後、AIエージェントが増えるインターネット社会において、本人確認という「信頼レイヤー」の構築はますます重要になると考えられ、Diditの取り組みはその一つのモデルとして注目に値します。

Y Combinator の 2026年冬（W26）バッチ参加企業の Didit が先月27日、シードラウンドで累計750万ドル（今回の追加調達分は600万ドル）を調達したことを発表した。 AI によるディープフェイクや […]



![Didit 創業者の双子兄弟 Alejandro Rosas 氏（CTO）と Alberto Rosas 氏（CEO）](/rss/downloaded_images/7b130b29eaee12846e2f44ca5784f978.jpg)



Y Combinator の 2026年冬（W26）バッチ参加企業の [Didit](https://didit.me/ja/) が先月27日、シードラウンドで累計750万ドル（今回の追加調達分は600万ドル）を調達したことを[発表した。](https://didit.me/ja/blog/didit-7-5m-seed-infrastructure-identity-fraud/)

AI によるディープフェイクや合成 ID の生成が容易になり、なりすましが現実の脅威となるなか、同社が掲げるのは「AI 時代のアイデンティティ・インフラ」だ。同社を本誌に紹介してくれたのは、福利厚生サービス「Fond」の創業者で、エンジェル投資家として Didit にも出資する福山太郎氏だ。

今回のラウンドには Y Combinator のほか、Pioneer Fund、Orange Collective、Founders Future、Phosphor Capital、SaaSholic、Rebel Fund、Lobster Capital などが参加。エンジェルとして、給与計算サービス「Gusto」共同創業者の Tomer London 氏や、前述の福山氏らが名を連ねる。調達資金はグローバルでの市場開拓、プログラマブルな本人確認・不正対策インフラの拡張、そしてプロダクト・営業・カスタマーサクセスの採用に充てる。

日本からもすでに利用のあるという Didit。CEO で創業者の Alberto Rosas 氏にメールインタビューを実施した。

### 双子のエンジニアが挑む「AIエージェント時代のKYC」

Didit は2023年、双子の兄弟でともに AI エンジニアの Alberto Rosas 氏（CEO）と Alejandro Rosas 氏（CTO）が創業したスタートアップだ。本人確認（KYC/KYB）から不正検知までを、開発者向けのひとつの API として提供する。

従来の本人確認サービスは、エンタープライズ向けの個別契約が前提で、国ごとに作り込まれた専用プロダクトが主流だった。営業と契約交渉を経て、数週間かけて統合する。Didit が作ったのはその真逆だ。開発者、あるいは AI エージェントが約5分で組み込めるセルフサーブ型のプロダクトを、グローバル規模で成立させた。

「本人確認と不正対策をひとつの API にまとめ、モジュールごとに価格を公開し、どんな開発者でも5分で実装できる、あるいは Claude Code や Codex、Cursor のような AI コーディングエージェントが、たった一度のプロンプトで実装できるほどシンプルにした」（Alberto 氏）。




![Didit のワークフロー作成画面（UI / API 両対応）](/rss/downloaded_images/857d3acf7063ab49559d67b7e8011c98.png)



その手応えは数字にも表れている。公表された情報によるとアクティブな B2B 顧客は1,500社を超え、提供地域は220カ国・地域に及ぶ。売上は前月比30%超で成長し、すでに黒字化を達成。

顧客の80%は、これまで本人確認サービスを導入していなかった新規層だという。1回の検証あたり200以上のシグナル（書類の真正性、生体活性、インジェクション攻撃やディープフェイクの解析、行動シグナルなど）を分析し、AI モデルはサードパーティに依存せず自社で内製している。

### 人的レビューは全体の5%

技術チームや AI エージェントに支持される理由のひとつが、確認フローを自由に設計できる柔軟性だ。閾値の設定、モジュールの有効化・無効化、どの警告で自動拒否とし、どの警告を手動レビューに回すか。これらをすべて API 経由で短時間に設定できる。

ここで気になったのは自動化と人手のバランスだ。

彼らのユーザーのグローバル平均で見ると、人的レビューに回るのは全体の約5%程度なのだそうだ。なかには分散型データマーケットプレイスのように、膨大な処理量を他の方法ではさばききれず、人的レビュー0%で運用する企業もあるという。

背景には、生成 AI がもたらした環境変化がある。

ディープフェイクや合成 ID、インジェクション攻撃の作成が容易になり、従来型の KYC では追いつかなくなりつつある。一方で規制も強化が進む。EU の eIDAS 2.0 は、全加盟国に2026年12月までのデジタル ID ウォレット提供を義務づけ、銀行・決済事業者には2027年12月までの受け入れを求めている。

### 多層アプローチによる信頼性の担保

Didit のアプローチは「層（レイヤー）」で考えるとわかりやすい。組み合わせるデータポイントが多いほど、不正をゼロに近づけられるという発想だ。

土台となるのが AI による書類検証だ。1万4,000種類以上の書類タイプ、220カ国以上に対応し、AI の OCR と偽造・改ざん検出に加え、対応書類によっては NFC チップの読み取りも行う。

その上に、パッシブ／アクティブの活性検知、1対1の顔照合、1対多の顔検索といった生体認証、さらにデバイスや行動・IP のシグナル、1,300以上の制裁・PEP（重要公人）・ネガティブ情報リストに対する AML スクリーニングを重ねる。

そして最上位に来るのが、政府データベースとの直接照合だ。

国民登録、税務当局、選挙人名簿といった一次情報源に対し、同社によると現在53カ国・159のデータソースで照合が可能だという。うち約30は政府レジストリへの直接連携で、アルゼンチンの RENAPER、ブラジルの Receita Federal、メキシコの RENAPO、ペルーの RENIEC、スペインの DNI 登録、インドの Aadhaar などが含まれる。

価格は1照合あたり0.05ドルからだ。Didit はオンボーディング時の本人確認にとどまらず、法人確認（KYB）や継続的なトランザクションモニタリングまでをカバーするプラットフォームだとしている。

### スペイン政府が「対面より安全」と認定、日本市場も視野



![](/rss/downloaded_images/963e9de87eacaacaaafef08fb82123b2.png)



220カ国超への展開を、同社はわずか数年で実現した。

その鍵となったのが、前述のセルフサーブ型というプロダクトモデルと、多層型の検証アプローチだ。

象徴的なのが、スペイン政府による認定である。

Didit は同国の金融サンドボックス（金融情報機関 SEPBLAC、証券規制当局 CNMV、財務省の監督下）において、NFC とアクティブ活性検知の組み合わせが「対面での本人確認と同等以上に安全」だと認定された唯一の本人確認プロバイダだという。Alberto 氏に尋ねたところ、約1年にわたる協働を経ての評価を得たという。

「私たちが本人確認から始めたのは、それが最も難しいピースだったからだ」と Alberto 氏はリリースでコメントしている。「本当に作ろうとしているのは、インターネットの信頼レイヤーそのものだ」。

日本市場についても、同社は強い手応えを感じているようだ。

メールでの取材に対し、Alberto 氏は「現在、日本ではすでに数十社の顧客がセルフサーブで私たちのプロダクトを使っており、とくに日本企業から大きな関心を寄せられている」と回答している。国際展開するフィンテック企業はグローバルな顧客をオンボーディングする必要があり、来年には日本にチームを構築するためのオフィスを開設する予定だという。

生成 AI の普及が、利便性と同時に「なりすまし」という新たなリスクを増幅させている。誰が、あるいは何が本物なのかを見極める仕組みは、人間・企業・AI エージェントが入り混じるこれからのインターネットで、ますます重要なインフラになっていく。本人確認という最も難しい領域から「信頼レイヤー」の構築に挑む Didit が、日本市場でどう広がっていくのか注目したい。
