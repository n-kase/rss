# 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?

*2026/6/15 RetroPC NEWS編集部*

**兵藤嘉彦さん**

1980年代、日本のパソコン黎明期から第一線で活躍してきたプログラマー、**兵藤嘉彦** (c.mos) さん。『**獣神ローガス**』『**ディガンの魔石**』やテキストエディタ『**VZ Editor**』を生み出しました。日本のパソコン史とともに歩んだ人生を語っていただきました。

**兵藤嘉彦さんプロフィール**

1980年代から活躍するプログラマーです。大学時代からパソコンに親しみ、1982年には野球ゲーム『中日－巨人戦』を月刊アスキーに掲載されました。その後、日立製作所勤務を経て、1987年には『獣神ローガス』の開発に参加。そして1988年、ビレッジセンターから発売されたテキストエディタ『VZ Editor』を世に送り出します。MS-DOS時代を代表する国産テキストエディタのひとつとして絶大な支持を獲得しました。

その後も、携帯ゲーム機WonderSwan向け開発環境『Wonder Witch』のBIOS開発や、視覚障害者向けソフトウェア、自動点訳システム、印刷業務管理システム、医療分野のシステム開発やIPA未踏ソフトウェア創造事業にも参加し、先進的なソフトウェア開発に携わりました。[https://x.com/c_mos](https://x.com/c_mos)



![2606_cmos_1-1 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/f90e936ea909f3dd15c2930e622bda4d.jpg)



**―― 兵藤さんがマイコンの世界に足を踏み入れたきっかけから教えてください。**

きっかけは父親の部屋に置いてあった『**マイコン入門**』や『**マイコンの使い方遊び方**』といった本でした。そこに映し出されていたグリーンモニターの画面を見て、なんだか無性にゾクゾクしたのを今でもよく覚えています。



![2606_cmos_DSC06258-1 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/1c7629eeb29f27e6e2a450e4c4f0c4af.jpg)



その後、大学に入学してから名古屋の大須にあるラジオセンターの2階へと足を運びました。そこに入っていたのが、秋葉原の本多通商の関連会社である「**中部本多通商**」でした。

お店の方に相談したところ、「**Z80はダメだ、6800の方が絶対にいいから！**」と熱っぽく勧められましてね（笑）。その言葉を信じて購入したのが、日立のスタータキット「**H68/TR**」でした。



![H68-TR 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/dff7e027b27b3167aa7a27bb03b26e46.jpg)



**――当時はまだワンボードマイコンの時代ですね。操作はいかがでしたか？**

アセンブラも電卓のようにキーボードから直接打ち込んでいました。その後、さらに「H68CTV1」というカラーモニターを接続するボードを追加しました。これによって、NECのTK-80BSのように通常のモニターに画面を出力できるようになりました。

## 人生を変えた「日立プログラムコンテスト」

**――その後はどのようなマシンに移られたのでしょうか。**

次に手にしたのが「ベーシックマスターL3（Basic Master Level 3）」でした。このマシンに搭載されていたCPU「6809」の命令体系が本当に素晴らしくて、すっかり魅了されてしまいました。

そんなある日、月刊I/O誌でプログラムコンテストが開催されることになったんです。私は**「ギャラクシアン」風のゲームを応募**して、見事に入選しました。



![2606_cmos_3-1 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/439f3ed155a33bc2654aef61ef3a1b7d.jpg)





![2606_cmos_2 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/545d1909f72b8c9cfba29bdc7fe9be5b.jpg)



**――素晴らしいですね！ その入選が大きな転機になったとか。**

有楽町でコンテストの発表会が開催されたのですが、そこで後にパソコンマニアの間で有名になる**「ZOB Plus」の速水祐さんと出会いました**。彼も、**「ギャラクシアン」風のゲームを応募**して入選したんですよ(笑)

さらに、この発表会が縁となって日立製作所へ入社しました。

## 日立入社、そして「S1」開発チームへ

**――入社後はどのような業務に就かれたのですか？**

3ヶ月の研修を終えた後、配属されたのはすでに結成されていた**「S1」の開発チーム**でした。

私が担当したのは、S1用の「**68008カード**」におけるCP/M-68K（16/32ビット向けのOS）の移植作業です。ただ、配属されたのがハードウェアチームだったのですが、当時の私は回路そのものにはあまり興味が持てなくて……。「今にして思えば、あの時もっと真面目に回路の勉強をしておけばよかったな」と、少し悔やまれますね。



![2606-cmos-68K-2 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/3f44a7675a42d1967e65ab4429612585.jpg)

**――当時のS1開発チームはどのような雰囲気だったのでしょうか。**

現場は主任の天野さん、そして後藤さん、兼八さんの3人だけで実務を回しているような非常に少数精鋭のチームでした。

ただ、残念ながらS1は商業的に大ヒットとはいかず、売れ行きは厳しかった。そのため、S1開発部自体がすぐに縮小・解散となってしまったんです。結局、私も2年ほどで日立を退職してしまいました。

## そしてFM-7への移植から始まった「勝手に作り直し」事件

**――日立を退職された後は、ランダムハウスに転職されたそうですね。**

速水さんとZOBのメンバーで、森田将棋などで有名な森田和郎さんの「**ランダムハウス**」を訪問したご縁で、**PC-8801版の『リグラス』をFM-7に移植することになりました**。プログラムは私とZOBの出口氏、テーマ曲は飯田氏の作曲です。



![2605_cmos_rogas2 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/8a70cdc3743a43ec045688dea1259991.jpg)



この時、はじめてZ80のコードを読んだのですが、6809と異なり、とても苦労しました。

FM-7の何が良かったかって、とにかく価格が安くて普及していたことですよね。ちなみに、同じCPU（6809）を積んでいるからといって「**S1とFM-7はCPU”しか”同じじゃないよ！**」と、何もかもが違いました笑。

その後、ランダムハウスに入って『**獣神ローガス**』の制作を担当することになります

**――そして、その『獣神ローガス』の開発で、語り草となるエピソードがあるとか。**

そうなんです。実は『**獣神ローガス**』を、私が勝手にゼロから作り直してしまったんですよ（笑）。当時はまだ、そういう大胆な進め方も許されちゃう大らかな時代でした。

元々は山口祐平さんが企画したSFアクションRPGで、開発中の画面がゲーム雑誌に載ったりもしていたんです。それを私が勝手にシミュレーションゲームの要素を入れて作り直してしまったものだから、発売が延期になってしまって。周囲からは「発売までに紆余曲折があって、**当初とは全然違うゲームが発売されることになった**」なんて言われてしまいました（笑）。



![2605_cmos_rogus 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/12ccdcdf264a25b2d3281c83129dfb96.jpg)



**――当時のPCゲーム界を代表するクリエイターの方々と、次々に濃いお仕事をされていますね。**

その後、山口さんが「アーテック」という会社を立ち上げて、スタジオぬえと一緒に『**ディガンの魔石**』というRPGの大作を作ろうと動いていました。

(* 画像提供:わんにゃーHashiさん*)



![2605_cmos_diga 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/2c50cadeb0d5c437dcb4d2e12f13ce17.jpg)



実は私は一度アーテックを離れていたのですが、開発に問題があり完成まで漕ぎ着けられないという状況になって、助っ人として戻ることになったんです。1ヶ月くらい開発に入りました。

## ビジネス機だと思っていたPC-9801が魅せた、圧倒的な可能性

**――PCゲーム黎明期を駆け抜けた兵藤さんですが、その後、PC-9801（以下、98）の世界に強く引き込まれていった理由は何だったのでしょうか。**

最大の理由は、とにかく圧倒的に「**性能がよかった**」ことです。

当時、ZOB Plusの速水さんのご自宅で、98用のゲームを見せてもらう機会がありました。**なかでもマーク・フリント氏が手掛けた作品の完成度**には、本当に大きな衝撃を受けましたね。それまで自分の中で98は「ビジネス用の堅い機械」というイメージだったのですが、「ここまで本格的で凄いゲームが動くのか！」と驚きました。



![2606_cmos_zone 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/ece53a111121f87f4a01bfff53106758.jpg)



PC-9801成功の要因として、CPU性能ではなく日本語環境の完成度が高かった。特に漢字対応テキストVRAMとノンインタレース表示を早期に採用したことで、当時の非力なCPUでも快適な日本語表示を実現しました。また、読みやすい半角英数字や漢字フォント、美しいNEC製モニタの表示品質にも魅力を感じていたといいます。さらに、ビジネス機と思われていたPC-9801で高度なゲームが動くことに衝撃を受け、その可能性を確信しました。日本語処理の快適さと高品質な表示環境が、PC-9801を成功へ導いた大きな要因だと思います。

**――そこから、ご自身での98向けソフトウェア開発が始まるわけですね。**



![2606_cmos_u2 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/4492d4ca4789225c50c172fe37e3718a.jpg)



初めて買った98は、**PC-9801U2**でした。これはFM音源はオプションでV30のクロックは8MHzでしたが、１年後に発売されたUV2はFM音源標準搭載でクロックも10MHz、悔しい思いをしました(笑)

当時は開発をしようにエディタがないと始まらないということで、『**EZ Editor**』を作り始めました。最初のバージョンのサイズが偶然9801バイトだったんです(笑)

残念ながら、もうこのコードは手元に残っていないんですけどね。

その後、速水さんのご紹介もあって、PC World Japanから『EZ Editor』を商業発売することになったんです。ところが、出版社の経営が厳しくて、印税の支払いが滞りがちでした。

## 伝説の『VZエディタ』誕生と、限界に挑んだアセンブラ開発

**――その苦い経験が、のちの歴史的名作『VZエディタ（以下、VZ）』の開発へとつながっていくのですね。**

そうですね。「それなら、改めて新しいエディタを作ろう」と開発したのがVZです。この時も速水さんから、ビレッジセンターの中村社長を紹介していただきました。



![manual_15 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/a06b3267c971899639434b62a79621f3.jpg)



当時はコマンドラインの編集やコンソールの出力結果を取り込むために、それぞれ別の常駐ソフト（TSR）をいくつも立ち上げるのが一般的でした。でも、「そんなの、エディタ本体に統合してしまえば不要な常駐ソフトを減らせるし、作業環境も圧倒的に効率化できるじゃないか」と考えたんです。

ただ、言うのは簡単ですが実現は一筋縄ではいきませんでした。特にVZをメモリに常駐させるにあたって、エディタというのは扱うテキスト量によって必要なメモリがその都度変化するという問題があります。



![2606_cmos_vz1a 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/6b9ec4a799ee0975b7eea6c045be209b.jpg)



**――パソコン通信（日経MIXやNifty-Serveなど）を通じて、多くのモニターから要望が寄せられたそうですが、機能の取捨選択には悩まされませんでしたか？**

そこが一番悩ましかったところです。モニターの皆さんからは本当にたくさんの熱心な要望が届きました。しかし、言われた通りにすべての機能を盛り込んでしまっては、ソフトがどんどん肥大化して、私が最も大切にしている「軽快さ」や「速度」が失われてしまいます。



![2606_cmos_DSC06270-2 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/cffec16b8f774a11983f4670ca121477.jpg)



そこで行き着いたのが、約80種類もの起動オプションと、あの強力なマクロ機能でした。「本体は極限までシンプルに保ち、必要な人が必要な機能だけを後から追加すればいい」という柔軟な設計です。

速水さんから『**WordStar』『Turbo C**』や『**Turbo Pascal**』といった海外の優れたツールを見せてもらい、そこで採用されていた「ダイヤモンドカーソル（Ctrlキーと特定のキーを組み合わせたカーソル移動）」などの操作体系もサポートしていきました。 (*画像はInternetArchiveより編集部加工*)



![2606_cmos_tp 【インタビュー】c.mos 兵藤嘉彦さん VZ Editor 生みの親 獣神ローガスを勝手に作り直した!?](/rss/downloaded_images/5e61f90b3329537b53fd829411e8a1f4.jpg)



**――ユーザーとのキャッチボールによって、VZに実装された機能も多かったのですね。**

例えば、当時はまだ未実装だった「**矩形（くけい）ブロック編集**」がそうでした。私自身はテキストを四角く切り取る必要性をあまり感じていなかったのですが、パソコン通信のモニターの皆さんから「どうしても欲しい！」と猛烈なプッシュがありまして（笑）。マクロで実現しました。

**――VZといえば、強力なカスタマイズ性と「マクロ機能」が多くのユーザーを熱狂させました。**

キー割り当てなどを変更できる設定ファイル（`vz.def`）は、**完全に速水さんの意見を取り入れて作ったものです**（笑）。他にも、競合ソフトの操作性を再現する`mifes.def`といった優秀な設定ファイルは、日経MIXやNIFTY-Serveといった当時のパソコン通信にいた、非常に詳しいヘビーユーザーの皆さんが作ってくれました。
