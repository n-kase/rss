概要と事実

世界中で電気をためる施設である蓄電所の建設コストが、ガスを使って電気をつくる火力発電所の新設コストを下回った。これはこれまでになかった初めての状況である。2025年における電池の価格は、前年と比べて約40％安くなった。この価格下落は、中国での電池の過剰生産と、電気自動車向けに作られていた電池を他の用途に転用できるようになったことが主な理由である。一方、ガス火力発電所を新設する際に必要なタービン（発電機の回転部品）の供給が追いつかなくなっており、その結果、ガス火力の建設コストが上がっている。

背景と結論

中東地域の不安定な状況がガスの燃料価格を押し上げており、これがガス火力のコスト上昇にさらに拍車をかけている。今後2026年以降、ガス価格が高止まりすれば、蓄電所とガス火力のコスト差はさらに広がると見込まれる。蓄電池は、天候に左右されやすい太陽光や風力などの再生可能エネルギーを安定して利用するために不可欠な技術であり、コストが下がることでこれらのクリーンエネルギーの普及が進むことが期待される。つまり、電池の価格低下とガス火力のコスト上昇が同時に進むことで、エネルギー供給の構造がより環境に優しい方向へシフトしつつあるということができる。
