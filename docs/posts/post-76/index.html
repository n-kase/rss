Interop Tokyo 2026の開催期間中に行われた「Internet x Space Summit」では、宇宙とインターネットの融合をテーマにさまざまな展示や講演が行われました。その中で、KDDI総合研究所はJAXAとの共同研究に基づいて開発中の、地球と月の間で光ビームを用いた衛星間光通信ターミナルの試作品をブースで公開し、Best of Show Awardのイノベーションテクノロジー（スペースチャレンジ）部門でグランプリを受賞しました。この技術は、約38万km離れた地球静止衛星と月周回衛星を光のビームで結ぶもので、極めて高い指向精度が求められ、手ブレ補正と追尾のための可動ミラーとセンサーシステムが核となっています。また、同じく summit 内では、月や火星での通信遅延を模したチャットデモや、衛星コンステレーションの追跡シミュレーション、災害時のStarlink利用を想定した防災通信キットなど、宇宙通信に関連するさまざまな展示が紹介されました。

会場での議論は、まず技術的な難易度に対する驚きと関心から始まりました。参加者は、横浜と舞浜の間で直径10cmのカップにホールインワンする精度に例えられる指向制御の難しさについて詳しく質問し、手ブレ補正アルゴリズムやセンサー融合の仕組みに関心を示しました。一方で、実用化に向けたコストや耐環境性、国際標準化の必要性について懸念する声もあり、時間が経つにつれて楽観的な展望と現実的な課題の両方が交錯する議論へと変化していきました。特に、月面基地や火星探査におけるリアルタイムデータ転送の可能性に期待が寄せられつつ、実際の運用に向けた試験環境の整備や法規制の整備が求められるという指摘が増えてきました。

このような技術の進展は、月や火星への人間の長期滞在を支える通信基盤としてだけでなく、地上での災害時通信のバックアップや、宇宙産業全体の競争力強化にもつながると考えられます。ネット上でも、KDDIの受賞を契機に「宇宙インターネット」への期待が高まり、官民連携や研究予算の増額を求める声が見られます。さらに、衛星画像の変化検出システム「DANGO」やスターリンクの防災キットなど、関連技術が同時に注目されることで、宇宙と地上の通信インフラが統合される未来への関心が高まっています。

したがって、今後は光通信の指向精度をさらに向上させる手ブレ補正技術の実証実験を継続し、国際的な標準策定に向けた協力体制を築くことが重要です。これにより、宇宙空間における信頼性の高いインターネット接続が現実となり、探査ミッションの成功だけでなく、地上の防災や産業革新にも貢献できるでしょう。です。

2026年6月15日 12:06

6月10日～12日に開催された「Interop Tokyo 2026」の中では、主催者企画として「Internet x Space Summit」を実施。宇宙とインターネットに関する展示や講演がなされた。

株式会社KDDI総合研究所のブースでは、JAXAとの契約にもとづいて研究開発している、月と地球の間の光通信の技術について展示していた。同展示は、Interop Tokyo出展者の製品を表彰する「Best of Show Award」にて、「イノベーションテクノロジー（スペースチャレンジ（宇宙産業））部門」のグランプリを受賞した。

今回の展示は、地球を回る静止衛星と、月を回る月周回衛星との間で光のビームにて通信するための技術だ。距離が38万kmあるため、ターゲットがごく小さく、例えるなら神奈川県の横浜と千葉県の舞浜の間で直系10cmのカップにホールインワンする精度が必要だという。そのための手ブレ補正と追尾の技術が今回の展示のテーマだ。

仕組みとしては、受信した光に対して、可動ミラーで“手ブレ補正”を行う。可動ミラーの先では光信号を、広視野光センサーと高感度光センサーに分光し、センサーの情報から可動ミラーを調整する。

なお、この技術の地上検証の開始について、2月18日付でプレスリリースとして発表されている。

また、同じく「Internet x Space Summit」のWIDE Project Space-WGのブースでは、地球・月・火星の間で、通信遅延時間を模して（月で1.3秒、火星で4分）チャットするシステムをデモしていた。

調査研究企業のDaiphys Technologies LLC（合同会社大物理技術研究所）のブースでは、NECとのコラボによる、衛星コンステレーションの打ち上げの追跡シミュレーションと、そのクイズを展示。また、衛星の画像を解析して時系列の変化を検出するシステム「DANGO」も展示していた。

株式会社インターネットイニシアティブ（IIJ）のブースでは、衛星通信のStarlinkのサービスとして、設置からのワンストップサービスや、防災通信キットを展示していた。
