スナック店舗向け DX プラットフォーム「スナテク」を運営するスナックテクノロジーズは8日、プレシリーズ A ラウンドで総額2億2,200万円を調達したことを公表した。出資者は X&KSK（共同創業者 本田圭佑氏）、セゾン・ベンチャーズ、ベクトル。今回の調達により、創業以来の累計調達額は3億5,600万円となった。

スナテクは、スナック店舗の運営をデジタル化する Web アプリ。会計・決済のキャッシュレス化、顧客・来店管理、料金明示、領収書自動発行といった機能を提供する。来店中ユーザーのリアルタイム表示や店内動画などの店舗ナビ機能も備えており、スナック経営者と来店客の双方を対象として展開している。

同サービスの特徴的な機能として、投げ銭連動のライブ演出機能「スナボム」がある。遠隔からの「デジタルおごり」で店内演出を再生する仕組みで、特許出願中となっている。

同社は2024年8月に設立。昨年2月には鈴木おさむ氏らが参加したシードラウンドで総額1億3,300万円を調達している。
