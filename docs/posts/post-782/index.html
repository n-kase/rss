概要と事実

WOWOWとNTTドコモは2024年6月16日に、ドコモの映像配信サービス「Lemino」を強化するための資本業務提携を発表しました。WOWOWは6月17日付で新会社を設立し、Leminoの事業をその新会社に吸収分割で移管します。ドコモは対価として新会社の株式49％を取得することになります。新会社は2024年10月1日に営業を開始予定です。これにより、WOWOWのコンテンツ制作力とドコモの会員基盤を組み合わせ、Leminoの競争力を高める狙いがあります。また、WOWOWはドコモに対して第三者割当増資を行い、調達した資金はLeminoのコンテンツ拡充や会員獲得のためのマーケティングに使われます。提携では音楽ライブの強化や超大作オリジナルドラマの制作、人気スポーツコンテンツの充実など、コンテンツの大幅な拡充を目指します。さらに、デジタルとリアルを融合した新しいエンターテインメント体験の提供、両社の顧客基盤やチャネルを活用したリーチの拡大、アーティストやクリエイターとの協業を通じたグローバルに展開できる知的財産の創出も掲げています。両社は2025年11月にコンテンツ分野の業務提携を結んでおり、音楽ライブやオリジナルドラマ、スポーツコンテンツの共同調達・共同制作・相互提供を進めてきました。今回の資本業務提携で関係を深め、Leminoの大幅な進化を図るとしています。

背景と結論

今回の提携は、映像配信市場が激化する中で、コンテンツの質と量を両方とも強化しようとする動きの一環です。WOWOWは長年にわたって映画やドラマ、音楽ライブなどの高品質なコンテンツを提供してきましたが、配信プラットフォームでの会員基盤は限定的でした。一方、ドコモは多数の携帯電話利用者を抱え、Leminoを通じて映像配信サービスを展開していますが、独自のコンテンツ制作力については課題がありました。両社がそれぞれの強みを掛け合わせることで、Leminoはより魅力的なラインナップを揃え、新規会員の獲得や既存会員の満足度向上を期待できます。さらに、音楽ライブやスポーツなどのリアルイベントと配信を結びつける取り組みは、デジタル時代におけるエンターテインメントの新しい形を示す可能性があります。結論として、この資本業務提携は両社の競争力を高め、利用者にとってより豊かな視聴体験を提供するための重要なステップとなるでしょう。

WOWOWはドコモに対し第三者割当増資を実施。調達した資金はLeminoのコンテンツ拡充や会員獲得マーケティングにあてる。

提携を通じ、音楽ライブの強化や超大作オリジナルドラマの制作、人気スポーツコンテンツの充実など、コンテンツを大幅に拡充する。
