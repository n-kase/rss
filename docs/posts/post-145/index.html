中古車の個人間売買プラットフォーム「カババ」を運営するアラカンは11日、シリーズ C ラウンドのセカンドクローズを実施し、累計調達額が15億円となったことを公表した。本ラウンドには名古屋テレビ・ベンチャーズ、りそなキャピタル、栖峰投資ワークス、愛知キャピタルが出資した。

カババは、専門の査定士が車両状態を評価し、相場情報とあわせて開示することで、個人間の中古車取引を安心かつ低コストで行えるプラットフォーム。

名義変更、車両運搬、保証までをネット上で完結できる点が特徴で、同社によると従来の流通コストを75％削減したという。公表している月間アクティブユーザー数は約120万人、累計 GMV は197億円、累計成約台数は約4,800台（2026年5月末時点）に達している。

調達資金は、営業・マーケティング・エンジニアなどの人材採用強化、広告クリエイティブとデータ基盤の強化を含む販売促進・認知向上施策、UI/UX 改善や AI を活用した査定・価格提示機能などの新規サービス開発に充当する。同社はこれにより、プラットフォームの利便性向上と利用者拡大を図るとしている。
