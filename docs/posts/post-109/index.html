AIによる要約

王将フードサービスは2026年3月期に売上高1168億円を記録し過去最高を達成したが、営業利益は前年比4.5％減少しているものの過去最高水準を維持している。

しかし、株価は2025年8月に3900円をピークに下落し、2026年3月から再び下落して現在2700円台で推移しており、投資家離れが進んでいると指摘されている。

この状況は、AI関連株が市場をけん引する中で非AI株である王将への投資意欲が低下し、資金流入が減少することで株価の低迷が続く可能性を示唆している。

業績は良好なのに市場評価が乖離すれば、資金調達コストの上昇や経営戦略への影響が懸念されるため、企業はDXやテクノロジー活用の具体的成果を示して投資家の理解を得る努力が必要であり、投資家側も業績と市場トレンドのバランスを考慮した判断が求められる。

「餃子の王将」を展開する王将フードサービスの2026年3月期における売上高は1168億円で、前年比5.2％の増収となり、過去最高を更新した。営業利益は同4.5％減少したが、過去最高を記録した前年度の水準をおおむね維持した。

しかし、同社の株価は低迷している。上昇続きだった株価は2025年8月に3900円を突破した後、10月頃から下落し始めた。12月は3200円台をキープしていたが、2026年3月から再び下落し、現在は2700円台を推移している。

AI関連株がけん引する昨今の株式市場では、比較的好調な銘柄でさえ、”非AI”株の場合は株価が下落する傾向がみられる。餃子の王将の業績も好調にみえるが、裏では「投資家離れ」が進んでいる。
