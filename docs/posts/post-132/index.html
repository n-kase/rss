概要と事実

Z.aiは2026年6月13日に、コーディングと長期エージェントタスクに特化した新しいAIモデル「GLM-5.2」を発表しました。このモデルは最大100万トークンのコンテキストウィンドウに対応し、一度の応答で最大128Kトークン（約131,072トークン）まで生成できます。現在のところ詳細なアーキテクチャやベンチマークスコアは公開されていませんが、前世代のGLM-5.1の強みを引き継ぎつつ性能を向上させた位置付けです。発表時点では、有料サブスクリプション「GLM Coding Plan」の利用者に提供が開始されており、従量制APIや一般向けチャットボット、Hugging FaceでのMITライセンスによるモデルウェイト公開は来週中に予定されています。GLM-5シリーズは2月にGLM-5、4月にGLM-5.1、そして今回のGLM-5.2と、わずか4か月で3世代目のアップデートとなります。前世代のGLM-5.1はSWE-Bench Proで58.4%を記録し、GPT-5.4やClaude Opus 4.6を上回っており、オープンウェイトモデルの中ではMiniMax M3やKimi K2.6と激しいスコア争いが続いています。したがって、GLM-5.2のベンチマーク次第ではシリーズ全体の首位を奪う可能性も示唆されています。

ネットの反応と意見の変遷

発表後、開発者コミュニティでは長いコンテキストを扱える点に期待が寄せられ、特に大規模なコードベースやドキュメントを一度に処理できる利便性が称賛されました。一方で、詳細な技術情報がまだ公開されていないため、実際の性能やリソース消費について慎重な見方をする声もありました。SNSでは、オープンソース化の予定が来週に控えていることへの喜びと、ライセンスがMITである点が広く利用しやすいと評価するコメントが目立ちました。また、ベンチマークでのスコア争いが激しい中で、GLM-5.2がどの程度の性能を示すか注目が集まっており、予想スコアが公開されればさらに議論が活発になると予測されています。全体としては、技術的進歩への期待と、実際の利用シーンでの検証待ちという二つの感情が混在している状況です。

背景と結論

今回の発表は、AIモデルの大規模化と特化が進む中で、コーディング支援や長文処理という実務ニーズに応える動きの一環です。長いコンテキストを扱えるモデルは、ソフトウェア開発におけるリファクタリングや大規模プロジェクトの理解を助け、開発者の生産性向上に寄与する可能性があります。また、オープンソース化によるMITライセンス採用は、研究者やスタートアップが自由にモデルを改変・利用しやすくすることで、イノベーションのスピードを加速させる期待があります。今後は、実際のベンチマーク結果やユーザーからのフィードバックが明らかになることで、GLM-5.2の真の価値が評価されるでしょう。これにより、AIを活用した開発支援ツールの競争がさらに激化し、より高性能で使いやすいモデルが普及していくことが予想されます。

100万トークン対応コーディングLLM「GLM-5.2」公開、来週オープンソース化

2026年6月13日 18:54

Z.aiは6月13日、コーディングと長期エージェントタスクに特化したAIモデル「GLM-5.2」を公開した。最大100万トークンのコンテキストウィンドウに対応し、1回の応答で生成できる出力トークン数は最大128K(131,072トークン)となる。GLM-5.2の詳細なアーキテクチャやベンチマーク数値は本稿執筆時点で公表されていないが、Z.aiは公式Xで、前世代の長期エージェントタスクへの強さとコーディング能力を継承したうえで強化したモデルだと位置付けている。

本稿執筆時点では、コーディング向けサブスクリプション「GLM Coding Plan」のプラン契約者向けに提供が開始されている。単発で利用できる従量制APIと一般向けチャットボット、Hugging FaceにおけるMITライセンスでのモデルウェイト公開は、いずれも来週中に開始する予定だ。

GLM-5.2は、コーディングと長期エージェントタスクに特化したAIモデル。GLM-5シリーズは、2月公開の「GLM-5」、4月公開の「GLM-5.1」と続き、わずか4カ月で3世代目の更新となる。GLM-5.2の詳細なアーキテクチャは公表されていないが、前世代のGLM-5.1は、総パラメータ7440億でアクティブパラメータ数400億のエキスパート混合モデル(MoE)アーキテクチャを採用している。

前世代のGLM-5.1はSWE-Bench Proで58.4%を記録し、GPT-5.4(57.7%)やClaude Opus 4.6(57.3%)を上回った。オープンウェイトモデルにおけるSWE-Bench Proのスコアは現在、MiniMax M3(59.0%)、Kimi K2.6(58.6%、最新のKimi K2.7 Codeのスコアは未公表)、GLM-5.1(58.4%)が僅差の激戦区となっており、GLM-5.2のスコア次第では同シリーズが首位を奪う展開も視野に入る。
