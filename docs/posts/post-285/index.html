| |

最近ジャンププラスで連載が始まって、現在第2話までが掲載されている「あっ、悪魔ちゃん」という漫画。

話題になっていたので読んでみたけれど、酷かった。

主人公の小学4年生の少女が片想いしている男子が父の単身赴任が終わって東京に定住する話が出ており、それを嫌がる主人公が悪魔のような奇抜な服装とメイクをして彼の前に現れ、興味を引く事で彼の上京を阻止しようとする話のようだ。

最悪だと思った。

え、なんで主人公は彼の上京を阻止しようとしてんの？彼にとってはどう考えても東京に行った方がいいでしょうに。まず主人公の思考に全く共感出来なくて困った。

東京に行けば、今の場所にいるよりも高い学歴を得る可能性が上がり、その分格段に将来の所得も上がるだろう。

少なくとも生涯所得に大きく関わる大学生までの若い時期は、絶対に東京に住んだ方がいい。娯楽が多いとか少ないとかのそういう些末な問題ではない。

勿論一番いいのは最初から東京（特に都心3区）で生まれる事だが、それでも小学生なら十分間に合う。ましてや成績優秀でコミュ力も高いイケメンならば小４から東京に行っても十分に間に合うだろう。中学受験の可能性もゼロではないし、それが無理でも高校や大学で挽回できる。

田舎で子育てするのは、虐待だよ。東京に生まれ育っただけで子供の可能性は無限に広がる。田舎で子育てする事はその可能性を絶つ事だ。

どんなにイケメンでも成績優秀でも田舎で暮らしていては進学にも就職にも限界があるんだから。

そういう意味では主人公もまた、田舎で子供を産んだ愚かな親の被害者ではあるのだけど、だからと言って同級生に対する行いを許せるかというと別の話。主人公のしている事は片想い相手の彼を狭い田舎に押し込め、未来を閉ざそうとする加害行為だ。

相手が片想い対象の男子ならば、なおのこと。結婚したら彼の収入は世帯収入に反映されて自分の人生にも関わるんだから、東京行きを応援する方が自然だろう。

それとも自分のような陰キャでおとなしい女子は人気者の彼には相手にされないと思い込んでいるから？だとしたらそれは彼が今の場所にいたままだったとしても同じだし、このまま中学高校と成長して他の女の子と付き合う様子を間近で見せ付けられる方が嫌だと思うんだけどなー。

これから方向性が大きく変わるならば別だが、少なくとも現段階では本当に最悪の漫画というほかはない。

漫画としてはコマ割りや構図も見やすくてよく出来ているんだが、それだけに根本的な設定の最悪さが勿体無い。

あんまり言いたくないけれどキャラクターの性別が男女逆なら絶対にこんなに好意的には受け止められず、フェミニストから叩かれて炎上してただろね。

特に舞台が広島じゃなくて九州の何処かだったらここぞとばかりにさす九さす九言われていて叩かれてと思う。

彼にとってはどう考えても東京に行った方がいいでしょうに。 そうかなあ 東京は人が住むところじゃねえよ 京都にしなさい京都に
