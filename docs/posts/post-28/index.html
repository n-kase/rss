2026年6月14日に行われたFIFAワールドカップ・グループFのオランダ対日本戦が2-2の引き分けに終わり、試合の実況や感想をX（旧Twitter）に連続で投稿したユーザーが多数いた。その結果、同年5月に導入された未認証アカウント向けの1日あたりの投稿上限（オリジナル投稿50件、リプライ200件）に達し、「一日のポスト上限に達した」といった報告が相次いだ。この制限は従来の2400件から大幅に引き下げられており、X側はバックエンドの負荷軽減を目的としているが、告知がほとんどなかったため多くのユーザーは制限に気づかずに投稿できなくなった状況に直面した。ワールドカップという世界的なリアルタイムイベントでは、無料ユーザーが特に制限に引っかかりやすく、一部のユーザーは有料のX Premiumへの加入を検討する動きも見られた。したがって、プラットフォーム側は利用制限の変更を事前に明確に伝える必要があり、ユーザーは自分の利用頻度に応じて投稿の頻度を調整するか、有料プランを検討するべきである。

日本にとって格上相手の対オランダ戦はSNSで大きな盛り上がりを見せた。X上では同点劇を伝える投稿が続いたが、その一方で「一日のポスト上限に達したそうです？そんなのあるっけ？」といった戸惑いの声や、サッカー関連アカウントが制限にかかったという報告が目立つ結果となった。

ユーザーが直面しているのは、Xが2026年5月に導入した未認証（無料）アカウント向けの投稿制限だ。X公式ヘルプ「Understanding X limits」によると、未認証アカウントが投稿できるのはオリジナルの投稿が1日50件、返信（リプライ）が1日200件まで。さらにこの上限は「30分単位（semi-hourly）」のより小さな枠に分割されており、短時間に投稿が集中すると、1日の合計に達する前でも一時的に投稿できなくなる仕組みになっている。

今回の制限が話題になっているのは、変更前との落差が大きいためだ。従来、1日あたりの投稿上限は2400件とされており、一般的な利用で意識することはほとんどなかった。また、今回の変更についてXは目立つ告知を出しておらず、2026年5月にヘルプページの記述が静かに書き換えられただけで、多くのユーザーは投稿できなくなって初めて変更に気づいたという経緯もありそうだ。

X側は制限の理由について、ヘルプページで「Xの裏側にかかる負荷を軽減し、ダウンタイムやエラーページを減らすため」と説明している。バックエンドの安定運用を主眼に置いた措置という位置づけだが、W杯のように世界中からリアルタイム投稿が殺到する大規模イベントでは、無料ユーザーを中心に制限へ引っかかりやすい状況が生まれている。

一方、有料のX Premiumに登録すると、この投稿数の上限は大幅に緩和される。料金は最も安いベーシックプランで月額368円（年額3916円）から。W杯期間中に制限を回避するため、課金するユーザーの動きも見られた。
