概要と事実

地球温暖化の影響で猛暑が年々厳しくなり、熱中症や既存の病気の悪化による死亡者が増えている。世界保健機関（WHO）は2026年6月11日にベルリンで新たな「熱中症対策行動計画（第2版）」を発表し、猛暑による死は避けられないものではないと主張した。WHOは熱中症が心血管疾患などの既存疾患を悪化させ、特に高齢者や乳幼児、心臓病・腎臓病を持つ人々に危険であると指摘している。一方で2024年の研究では、猛暑による死亡者は高齢者よりも若者が圧倒的に多いことが示されており、屋外で働く農業や建設業の従事者、スポーツをする若者が影響を受けやすいと考えられている。ヨーロッパでは特に温暖化の進行が速く、過去4年間で20万人以上が熱中症で亡くなったと、WHO欧州地域事務局長のハンス・ヘンリ・クルーゲ博士は説明している。ドイツの連邦環境・気候対策・自然保全・原子力安全大臣カルステン・シュナイダー氏も、気候変動による猛暑が社会的課題であり、都市部のコンクリートに覆われた住環境では暑さから身を守ることが難しいと述べている。

背景と結論

猛暑による健康被害は、気候変動が進行する中で誰もが無関係ではいられない問題となっている。WHOの新しい行動計画は、地方自治体や関係機関が熱中症予防のための具体的な対策を立てやすくすることを目的としており、早期警戒システムの整備、冷房施設へのアクセス改善、啓発活動の強化などが含まれる。これにより、特に脆弱な高齢者や若年層の死亡リスクを低減できる可能性がある。同時に、都市計画における緑地の確保や建物の断熱性能向上など、長期的な適応策も求められている。今回の発表は、熱中症による死亡は防げるという認識を広め、政策や個人の行動変容を促すきっかけとなることが期待される。
