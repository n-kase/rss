キャラクターを活用した行動変容プラットフォーム「キャラミクス」を開発するリモアは4日、1億円の資金調達を実施したことを公表した。出資者は Headline Asia。

キャラミクスは、睡眠を起点にキャラクターによる接客、ID 連携、生活データ連携、パーソナライズ提案を統合し、ユーザーの継続行動を支援する行動変容プラットフォーム。同社はこれを「Character-Driven Behavior OS」と位置づけている。

技術面では、フル LLM による自由会話ではなく IP や世界観を守る制御設計を前提とし、感情設計とインセンティブ設計を組み合わせた設計を採用している。SDK/API として外部サービスへの組み込みを想定しており、美容、金融、EC、ウェルネスを中心に企業や自治体への展開を計画している。同社によると、美容・小売領域を中心に大手企業を含む複数社との PoC を推進中という。

調達資金はプロダクト開発、AI 開発、コンテンツ開発、人材採用の強化に充当する。今後のロードマップとして、2026年後半にキャラミクスの正式リリースおよび企業・自治体向け実運用導入を開始し、2027年以降は金融、EC、美容、教育、ウェルネス領域への横断展開と、Charamix ID を基盤とした BtoBtoC モデルの本格展開を予定している。
