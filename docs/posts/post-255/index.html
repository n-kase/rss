AIによる要約

概要と事実

南アフリカ共和国のケープタウン近郊にあるAquila Private Game Reserveで、夫婦がプールサイドで日光浴をしていた。二人は手元のスマートフォンに夢中になっており、すぐ後ろから近づいてきたチャクマヒヒ（オナガザル科の大型ヒヒ）に気付かなかった。ヒヒはデッキチェアの間に置いてあったスナックや果物を食べ始め、やがて女性がヒヒの存在に気付いて目が合った。男性も気付いたが、至近距離にいたため固まってしまった。ロッジのスタッフが駆けつけた際にヒヒは逃げ去り、夫婦は無事だった。

ネットの反応と意見の変遷

ネット上では、夫婦のスマホ依存を指摘する声が多数上がった。「わざわざ南アフリカまで行ってやることがスマホって…」や「現代人ならではだなあ」といった批判的なコメントが目立った。一方で、危険な状況にもかかわらず悲鳴を上げず手を出さなかった夫婦を「危機管理が出来ていて偉い」と評価する意見も見られた。また、ヒヒの行動や人間側の立場について「動物保護区内では人間が後からお邪魔している」という指摘や、服を着ていない状況での遭遇は固まるしかないという感想も共有された。時間が経つにつれて、単なる笑い話としてだけでなく、スマホ使用の危険性や自然への敬意を考えるきっかけとして捉える声も増えていった。

背景と結論

この保護区は柵がなくライオンやゾウなどが自由に歩き回る環境で、人間は動物の住処に後から入り込んでいる立場である。チャクマヒヒは知能が高く、食べ物を見つけると積極的に近づく習性があるため、人間側が注意を払わないとこうした接触が起こり得る。今回の出来事は、スマホに没頭していると周囲の状況に気付きにくくなるという現代社会の課題を如実に示している。自然の中で本当に体験すべきは画面の中ではなく、実際の風景や動物の振る舞いであり、そうした瞬間を逃さないためにも、デジタル機器から適度に距離を置くことが重要であると結論付けられる。

夫婦は画面を凝視していたため、最初は真横に置いてあるスナックをヒヒが食べているのに全く気が付かなかった。

ようやく女性が気付いてヒヒと目を合わせた。男性も気が付いたが、あまりにも至近距離にいたため、どうしていいのか固まってしまったようだ。

南アフリカの大自然を訪れながらスマホに没頭する夫婦

南アフリカ共和国のケープタウン近郊にあるAquila Private Game Reserveは、約10,000ヘクタールもの広大な面積を持つ私営の野生動物保護区だ。

敷地内には人間と動物を隔てる柵がなく、ライオンやアフリカゾウなど、野生動物が自由に歩き回っている。

敷地内に併設された宿泊施設、サファリロッジのプールサイドで、ある男女の夫婦が予想もしないハプニングに見舞われた。

夫婦はせっかく大自然とを満喫する機会を得たにもかかわらず、デッキチェアに横たわって手元のスマートフォンの画面を見入っていた。

そのため、すぐ後ろから1頭の大きな影が近づいていることに全く気がつかなかった。

至近距離までヒヒが接近、夫婦は硬直

夫婦の背後にく現れたのは、現地に生息するオナガザル科のチャクマヒヒ（Chacma baboon / 学名：Papio ursinus）だ。

チャクマヒヒはヒヒのなかでも大きく、成長したオスは体重が30kgから40kg近くに達し、鋭い犬歯を持つため非常に力強い。

そのヒヒが夫婦のデッキチェアの間においてあるサイドテーブルのスナックや果物を食べ始めたのだ。

そもそも、宿泊施設やプールがあるエリアはもともとチャクマヒヒたちの住処であり、人間側が後からお邪魔して滞在させてもらっている立場にある。

チャクマヒヒは極めて知能が高くて環境への適応能力に優れているため、ロッジ内で食べ物を見つけると絶好のチャンスとばかりに食べにくるという。
