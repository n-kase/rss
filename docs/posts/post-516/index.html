概要と事実

Appleは2024年6月18日に、Apple Vision ProのApp Storeが新しい市場へ拡大すると発表しました。中国本土、香港、日本、シンガポールでは6月28日から、オーストラリア、カナダ、フランス、ドイツ、イギリスでは7月12日からサービスが開始されます。開発者はApp Store Connectで選択した地域に対して、自分のvisionOSアプリおよび互換性のあるiPhoneまたはiPadアプリの提供可否をいつでも管理できます。また、アプリをApple Vision Proで直接評価してもらうリクエストを出したり、製品ページのメタデータを現地のユーザーに合わせてローカライズしたりすることも可能です。さらに、visionOSの機能を最大限に活用するネイティブアプリの作成方法について学ぶリソースも提供されています。

背景と結論

今回の展開は、空間コンピューティングプラットフォームであるVision Proのエコシステムを世界中に広げる狙いがあります。新たな地域での利用可能になることで、開発者はより多くのユーザーにアプリを届けやすくなり、市場の拡大が期待されます。また、ローカライズや評価リクエストなどのサポートを提供することで、地域ごとのニーズに合わせたアプリ改善が促進され、全体的なユーザー体験の向上にもつながると考えられます。このように、Appleはハードウェアの発売だけでなく、開発者支援と市場拡大を同時に進めることで、Vision Proの普及を加速させようとしています。
