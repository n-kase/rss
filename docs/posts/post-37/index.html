今あなたが使っているミュトス級AI「Claude Fable 5」、22日を過ぎたらどうなる？--発表を整理

2026年6月15日月曜日 1 minute read

概要（何が起きているのか）

  • Anthropicが6月9日に最上位AIモデル「Claude Fable 5」を提供開始し、有料サブスクリプションユーザーはすぐに利用可能になった。
  • 同モデルの無償提供は6月22日で終了し、23日以降は別途「利用クレジット」を購入しないと使えなくなる。
  • Opus 4.8以下の通常モデルはこれまで通り定額プラン内で利用できるまま。

示唆（それによって何が引き起こされそうなのか）

  • 6月23日以降にFable 5を使い続けると追加費用が発生し、予想外の出費に驚くユーザーが増える可能性がある。
  • ネットでは「コスパが悪くなる」「他のモデルに乗り換えるべき」といった声や、容量が確保次第復活するとの期待が混在している。
  • Anthropicは一時的措置だと説明し、容量に余裕ができたら再び定額プランに戻す意向を示している。

まとめ（だからどうなのか）

  • Fable 5を引き続き使いたい場合は、6月22日までに利用クレジットを購入するか、使用頻度を見直して予算内に収めるようにする。
  • コストを抑えたい場合は、 Opus 4.8 など従来モデルに切り替えて、追加料金なく利用を続ける。

元スレッド・記事本文

米Anthropicが6月9日（現地時間）に提供を開始した最上位AIモデル「Claude Fable 5」。Claudeの有料サブスクリプションユーザーはすでに利用可能になっているが、一方でAIモデル選択時の「6月22日まで含まれます」という注釈が気になっているユーザーも多いだろう。

Anthropicの発表によれば、Fable 5の有料プランへの追加料金なしでの提供は6月22日で終了となる。6月23日以降は同モデルをいったん各プランから外し、利用には別途「利用クレジット」の購入が必要になる。つまり、有料サブスクリプションだけでは利用できなくなる。

　具体的には、6月23日以降にFable 5を使う場合は、プランの利用枠とは別建ての利用クレジットを消費する従量課金となる。なお、Opus 4.8以下の通常モデルは従来通りプラン内で利用できる。

Fable 5は、従来の最上位だったOpusクラスの上に位置する新階層「Mythos級」初の一般提供モデルであり、API価格は100万トークンあたり入力10ドル、出力50ドルで、Opus 4.8の2倍に設定されている。

プランから一時除外する理由について同社は、「Fable 5への需要は非常に高く、予測が難しい」と説明。APIと従量課金型のEnterpriseプランでは当初から全面提供する一方、定額のサブスクリプションプランでは段階的に展開する方針を取ったとしている。容量に余裕があれば、追加料金なしの期間を22日以降に延長する可能性にも言及した。

あくまで一時的な措置との位置付けで、同社は「十分な容量を確保でき次第、Fable 5をサブスクリプションプランの標準的な一部として復活させることを目指す。できる限り早くそうするつもりだ」とし、変更がある場合は事前に告知するとしている。
